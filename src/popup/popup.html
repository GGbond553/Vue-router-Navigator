Vue-Router-Navigator

v0.5.0 - 本地开发路由导航工具
📋 使用规则
  • 仅在 localhost/127.0.0.1 生效
  • 页面加载后自动检测 Vue Router
  • 拖拽悬浮窗调整位置
  • 使用 Alt+R 快捷键聚焦搜索
✨ 核心功能
自动嗅探页面使用的 vue-router 实例，提供一键式、可视化、可拖拽的路由导航体验。
🔍 智能路由检测
🎯 一键跳转
📋 多级菜单
🎨 纯黑白灰设计
🖱️ 可拖拽悬浮
⌨️ 快捷键支持
🚀 高级特性
• 实时搜索功能，支持Enter键快速跳转
• 智能边缘检测，防止悬浮窗超出屏幕
• 位置记忆功能，刷新后自动恢复位置
• 平滑动画过渡，提供流畅交互体验
仅在本地开发环境生效，安全可靠 • 支持Vue2/Vue3
